# 💼 Smart Business Management System

A desktop-based application developed using Python to help small businesses manage their daily operations efficiently.

---

## 🚀 Features

- 🔐 Secure Login & Registration System  
- 📦 Product Management (Add & View Products)  
- 👥 Customer Management  
- 💰 Sales Entry with Automatic Calculation  
- 📉 Sales Analysis using Graphs  
- 📊 Dashboard with Business Insights  

---

## 🛠️ Technologies Used

- Python  
- Tkinter & ttkbootstrap (GUI)  
- MySQL (Database)  
- Pandas (Data Analysis)  
- Matplotlib (Graph Visualization)  

---

## 🧠 Project Overview

This system is designed to replace manual business management processes with a digital solution. It allows business owners to manage products, customers, and sales in an organized way.

The system automatically updates stock after each sale and provides graphical analysis of sales data.

---

 Setup Instructions

1. Clone the repository:

git clone https://github.com/your-username/smart-business-management-system.git


2. Import database:
- Open MySQL Workbench  
- Run the `smart_business_db.sql` file  

3. Update database connection in `database.py`:

host="localhost"
user="root"
password="your_password"
database="smart_business_db"


4. Install required libraries:

pip install pandas matplotlib ttkbootstrap


5. Run the project:

python main.py
